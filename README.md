
# Planet Clear (Salve a terra) ![img.png](img.png)

---

## Aplicação que remunera seus usuários após a realização de uma tarefa ligada a limpeza e conservação do meio ambiente.

### Perguntas frequentes:

**Qual problema o software resolve** : Poluição humana.

**Como o software vai resolver o problema** : Encontrando uma ação(Tarefa) para  usuário(Ativista) executar, com o objetivo
de ajudar o  meio ambiente e consequentemente a terra.

**E quais são essas tarefas** : Plantar árvores em áreas desmatadas, em parques ou áreas comuns das cidades,
reciclagem, coleta de lixo em rios, córregos, estradas, riachos, nascentes, campanhas de concientização e etc.
